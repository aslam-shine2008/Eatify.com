# Food Partner Website

## 📌 Project Description
Food Partner Website is a responsive restaurant menu website developed using HTML, CSS, and JavaScript. It allows users to browse restaurant partners and view their menu cards in a clean and user-friendly interface.

## 🚀 Features
- Responsive design
- Restaurant partner cards
- Individual menu pages
- Smooth navigation
- Mobile-friendly layout
- Attractive UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
